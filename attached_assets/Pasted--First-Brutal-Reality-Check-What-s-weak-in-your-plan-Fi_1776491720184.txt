🔹 4. 🔥 THE CORE: Scoring Engine (THIS WINS)

This is where you beat everyone.

Build a weighted scoring system:
score =
  + (type_match * 30)
  + (skill_match * 25)
  + (eligibility_match * 20)
  + (deadline_urgency * 15)
  + (location_match * 5)
  + (completeness_score * 5)
🔹 Key Logic
✅ Type Match
Internship matches → +30
Not preferred → +5
✅ Skill Matching (VERY IMPORTANT)
Compare extracted requirements vs student skills
% match → scaled score
✅ Eligibility Check
If CGPA/semester fails → BIG penalty
✅ Deadline Urgency
days_left = deadline - today

if days_left <= 3 → +15
elif <= 7 → +10
elif <= 14 → +5
else → +2
✅ Completeness
Has deadline + link + requirements → higher score
