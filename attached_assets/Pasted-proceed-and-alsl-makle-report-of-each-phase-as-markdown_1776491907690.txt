🧠 The Correct Approach (and why)
❌ Don’t let AI decide priority

If you ask the model:

“Rank these opportunities”

You’ll get:

Different rankings for same input
No clear scoring logic
Weak explanations (“seems relevant…”)

👉 Judges can easily ask:

“Why is this ranked #1?”

And you won’t have a solid answer.

✅ What you SHOULD do
🔥 Hybrid System (This is the winning design)
1. AI → Extract + Interpret

AI’s job:

Detect if it's an opportunity
Extract structured data
Understand messy text

Example output:

{
  "type": "internship",
  "deadline": "2026-05-01",
  "skills_required": ["Python", "ML"],
  "eligibility": "CGPA > 3.0",
  "location": "Remote"
}
2. Python Backend → Priority Scoring (CORE LOGIC)

👉 YOU control this part

This is where your intelligence shows.

score = 0

# Type match
if opp["type"] in user["preferred_types"]:
    score += 30

# Skill match
match = len(set(opp["skills_required"]) & set(user["skills"]))
score += match * 10

# Deadline urgency
days_left = (deadline - today).days
if days_left <= 3:
    score += 20
elif days_left <= 7:
    score += 10

# Eligibility
if user["cgpa"] >= required_cgpa:
    score += 20
else:
    score -= 50
